# Country Explorer Web App

## Description
A responsive web application that allows users to search for countries and view key details such as capital, population, region, subregion, and languages. T

## Features
- Search countries by name, even wiht incomplete name
- View detailed country information in a modal
- Responsive card-based UI
- Dynamic data fetching using public APIs
- Food and cuisine integration,filter by region (in progress)

## Tech Stack
- HTML
- CSS (Tailwind CSS)
- JavaScript (Vanilla JS)
- REST Countries API
- TheMealDB API

## How to Use
1. Enter a country name in the search bar
2. Click the search button or press Enter
3. Click on a country card to view detailed information
4. Use the region filter to narrow down results

## Current Status
- Country data fully functional
- Region filtering is partially implemented and under development
- Food/cuisine feature partially implemented and under development
